The Loni Panchayat Tax Management System is a modern, cloud-based web 
application designed to streamline property tax administration for Gram 
Panchayats in Madhya Pradesh, India. It replaces traditional manual processes 
with an automated, efficient, and transparent digital solution.

TARGET USERS:
• Panchayat Administrative Staff
• Panchayat Secretaries
• Tax Collection Officers
• Property Owners (bill recipients)

• Professional government-standard receipts
• Transparent revenue tracking and reporting
• Improved citizen satisfaction

4.3 TAX CALCULATION ENGINE
--------------------------
Tax Types:
1. Property Tax (per sq.ft rates):
   - Urban: ₹1.50 per sq.ft
   - Semi-Urban: ₹1.00 per sq.ft
   - Rural: ₹0.50 per sq.ft

2. Water Tax (annual):
   - Residential: ₹300
   - Commercial: ₹800
   - Industrial: ₹1,000
   - Agricultural: ₹100

3. Sanitation Tax (annual):
   - Residential: ₹200
   - Commercial: ₹600
   - Industrial: ₹800
   - Agricultural: ₹50

4. Lighting Tax (annual):
   - Residential: ₹150
   - Commercial: ₹500
   - Industrial: ₹600
   - Agricultural: ₹30

Calculation Logic:
• Property Tax = Area × Rate (location-based)
• Other taxes = Fixed annual amount (property type-based)
• Early Payment Discount = 5% if paid before March 31
• Late Payment Penalty = 1.5% per month after due date

Assessment Year:
• Current year (e.g., 2025-2026)
• Historical records maintained for all years

7.3 SECURITY RULES
------------------
Firestore Rules:
```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    
    // Helper function to check if user is authenticated
    function isAuthenticated() {
      return request.auth != null;
    }
    
    // Helper function to check if user is admin
    function isAdmin() {
      return isAuthenticated() && 
             get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    
    // Properties collection
    match /properties/{propertyId} {
      allow read: if isAuthenticated();
      allow create: if isAuthenticated();
      allow update: if isAuthenticated();
      allow delete: if isAdmin();
    }
    
    // Taxes collection
    match /taxes/{taxId} {
      allow read: if isAuthenticated();
      allow write: if isAuthenticated();
    }
    
    // Payments collection
    match /payments/{paymentId} {
      allow read: if isAuthenticated();
      allow create: if isAuthenticated();
      allow update: if isAdmin();
      allow delete: if isAdmin();
    }
    
    // Users collection
    match /users/{userId} {
      allow read: if isAuthenticated();
      allow write: if isAdmin();
    }
    
    // Settings collection
    match /settings/{settingId} {
      allow read: if isAuthenticated();
      allow write: if isAdmin();
    }
  }
}
```

8.2 FIRESTORE DATABASE API
--------------------------

Create Property:
```typescript
import { collection, addDoc } from 'firebase/firestore';

const propertyData = {
  ownerName: "John Doe",
  fatherName: "James Doe",
  mobileNumber: "9876543210",
  // ... other fields
  createdAt: new Date(),
};

await addDoc(collection(db, 'properties'), propertyData);
```

Read Properties:
```typescript
import { collection, getDocs, query, where } from 'firebase/firestore';

const q = query(
  collection(db, 'properties'),
  where('propertyType', '==', 'Residential')
);

const querySnapshot = await getDocs(q);
querySnapshot.forEach((doc) => {
  console.log(doc.id, doc.data());
});
```

Update Property:
```typescript
import { doc, updateDoc } from 'firebase/firestore';

await updateDoc(doc(db, 'properties', propertyId), {
  mobileNumber: "9999999999",
  updatedAt: new Date(),
});
```

Delete Property:
```typescript
import { doc, deleteDoc } from 'firebase/firestore';

await deleteDoc(doc(db, 'properties', propertyId));
```

Real-time Listener:
```typescript
import { collection, onSnapshot } from 'firebase/firestore';

const unsubscribe = onSnapshot(
  collection(db, 'properties'),
  (snapshot) => {
    snapshot.forEach((doc) => {
      console.log(doc.id, doc.data());
    });
  }
);

// Cleanup
unsubscribe();
```

12.1 COMMON ISSUES
------------------

Issue: Cannot Login
Symptoms: Login page shows error
Causes:
• Wrong email/password
• Internet connection issues
• Firebase service down
• Browser cache issues

Solutions:
1. Verify credentials
2. Check internet connection
3. Clear browser cache and cookies
4. Try different browser
5. Reset password if forgotten
6. Contact admin if user deactivated

---

Issue: Property Not Saving
Symptoms: Registration form shows error
Causes:
• Required fields missing
• Invalid data format
• Database connection issue
• Permission error

Solutions:
1. Fill all required fields (marked with *)
2. Verify mobile number (10 digits)
3. Check area is positive number
4. Refresh page and try again
5. Check console for errors
6. Contact support if persists

---

Issue: PDF Not Generating
Symptoms: Download button not working
Causes:
• Missing property data
• Browser popup blocker
• JavaScript error
• PDF library issue

Solutions:
1. Verify property has all data
2. Disable popup blocker
3. Try different browser
4. Check browser console
5. Clear cache and reload
6. Contact support with error details

---

Issue: Slow Performance
Symptoms: Pages loading slowly
Causes:
• Poor internet connection
• Large number of records
• Database not optimized
• Too many concurrent users

Solutions:
1. Check internet speed
2. Close unnecessary tabs
3. Clear browser cache
4. Use filters to reduce data
5. Archive old records
6. Upgrade Firebase plan

---

Issue: Data Not Syncing
Symptoms: Changes not appearing
Causes:
• Internet connection lost
• Firebase sync disabled
• Browser offline mode
• Service worker issue

Solutions:
1. Check internet connection
2. Refresh page (Ctrl+F5)
3. Clear service worker cache
4. Disable browser offline mode
5. Check Firebase status page
6. Restart browser

12.2 ERROR CODES
----------------

AUTH001: Invalid Login Credentials
• Cause: Wrong email or password
• Solution: Verify credentials or reset password

AUTH002: User Not Authorized
• Cause: User deactivated or insufficient permissions
• Solution: Contact admin for access

DB001: Database Connection Failed
• Cause: Internet issue or Firebase down
• Solution: Check connection, try again later

DB002: Write Operation Failed
• Cause: Permission denied or validation error
• Solution: Check user permissions and data validity

PDF001: PDF Generation Failed
• Cause: Missing data or library error
• Solution: Verify all property data exists

CALC001: Tax Calculation Error
• Cause: Invalid input parameters
• Solution: Verify area, rates, and property type
