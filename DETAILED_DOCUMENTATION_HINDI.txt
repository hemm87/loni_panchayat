                         1. परिचय
================================================================================

उद्देश्य:
लोनी पंचायत कर प्रबंधन प्रणाली एक आधुनिक, क्लाउड-आधारित वेब एप्लिकेशन है 
जो मध्य प्रदेश, भारत में ग्राम पंचायतों के लिए संपत्ति कर प्रशासन को 
सुव्यवस्थित करने के लिए डिज़ाइन किया गया है। यह पारंपरिक मैनुअल प्रक्रियाओं 
को एक स्वचालित, कुशल और पारदर्शी डिजिटल समाधान से प्रतिस्थापित करता है।

लक्षित उपयोगकर्ता:
• पंचायत प्रशासनिक कर्मचारी
• पंचायत सचिव
• कर संग्रह अधिकारी
• संपत्ति मालिक (बिल प्राप्तकर्ता)

सिस्टम लाभ:
• मैनुअल प्रोसेसिंग समय में 80% की कमी
• गणना त्रुटियों का उन्मूलन
• संपत्ति और कर डेटा तक रियल-टाइम पहुंच
• व्यावसायिक सरकारी मानक रसीदें
• पारदर्शी राजस्व ट्रैकिंग और रिपोर्टिंग
• बेहतर नागरिक संतुष्टि

4.1 उपयोगकर्ता प्रमाणीकरण
------------------------
विशेषताएं:
• ईमेल/पासवर्ड प्रमाणीकरण
• Firebase Auth के साथ सुरक्षित लॉगिन
• पासवर्ड रिकवरी (रीसेट)
• सत्र प्रबंधन
• निष्क्रियता पर ऑटो-लॉगआउट

कार्यान्वयन:
- Firebase Authentication SDK
- Next.js मिडलवेयर का उपयोग करके संरक्षित रूट
- भूमिका-आधारित पहुंच (व्यवस्थापक/उपयोगकर्ता)

4.2 संपत्ति प्रबंधन
-----------------------
पंजीकरण फ़ील्ड:
• मालिक का नाम (आवश्यक)
• पिता का नाम (आवश्यक)
• मोबाइल नंबर (10 अंक, आवश्यक)
• पता (आवश्यक)
• संपत्ति ID (स्वतः जनरेट)
• मकान नंबर (वैकल्पिक)
• संपत्ति प्रकार (आवासीय/वाणिज्यिक/औद्योगिक/कृषि)
• वर्ग फुट में क्षेत्रफल (आवश्यक)
• स्थान वर्गीकरण (शहरी/अर्ध-शहरी/ग्रामीण)

संचालन:
• नई संपत्ति बनाएं
• संपत्ति विवरण देखें
• संपत्ति जानकारी अपडेट करें
• संपत्ति हटाएं (केवल व्यवस्थापक)
• मालिक के नाम, संपत्ति ID, मोबाइल द्वारा खोजें
• संपत्ति प्रकार, स्थान द्वारा फ़िल्टर करें
• संपत्ति सूची निर्यात करें (भविष्य)

सत्यापन नियम:
• मोबाइल: बिल्कुल 10 अंक
• क्षेत्रफल: केवल सकारात्मक संख्या
• सभी आवश्यक फ़ील्ड भरे जाने चाहिए
• अद्वितीय संपत्ति ID जनरेशन

4.3 कर गणना इंजन
--------------------------
कर प्रकार:
1. संपत्ति कर (प्रति वर्ग फुट दरें):
   - शहरी: ₹1.50 प्रति वर्ग फुट
   - अर्ध-शहरी: ₹1.00 प्रति वर्ग फुट
   - ग्रामीण: ₹0.50 प्रति वर्ग फुट

2. जल कर (वार्षिक):
   - आवासीय: ₹300
   - वाणिज्यिक: ₹800
   - औद्योगिक: ₹1,000
   - कृषि: ₹100

3. स्वच्छता कर (वार्षिक):
   - आवासीय: ₹200
   - वाणिज्यिक: ₹600
   - औद्योगिक: ₹800
   - कृषि: ₹50

4. प्रकाश कर (वार्षिक):
   - आवासीय: ₹150
   - वाणिज्यिक: ₹500
   - औद्योगिक: ₹600
   - कृषि: ₹30

गणना तर्क:
• संपत्ति कर = क्षेत्रफल × दर (स्थान-आधारित)
• अन्य कर = निश्चित वार्षिक राशि (संपत्ति प्रकार-आधारित)
• शीघ्र भुगतान छूट = 31 मार्च से पहले भुगतान पर 5%
• विलंब भुगतान शुल्क = नियत तिथि के बाद प्रति माह 1.5%

मूल्यांकन वर्ष:
• वर्तमान वर्ष (उदा., 2025-2026)
• सभी वर्षों के लिए ऐतिहासिक रिकॉर्ड बनाए रखे गए

4.4 बिल जनरेशन
-------------------
PDF संरचना:
• पंचायत नाम और लोगो के साथ शीर्षलेख
• बिल संख्या (RCT-YYYYMMDD-XXXXX प्रारूप)
• बिल तिथि और मूल्यांकन वर्ष
• संपत्ति मालिक विवरण बॉक्स
• संपत्ति जानकारी (ID, प्रकार, क्षेत्र, स्थान)
• कर विभाजन तालिका:
  - कर प्रकार
  - मूल्यांकन वर्ष
  - आधार राशि
  - स्थिति (भुगतान/लंबित)
  - कुल राशि
  - भुगतान की गई राशि
• भुगतान सारांश:
  - कुल देय
  - भुगतान की गई राशि
  - शेष बकाया
• भुगतान निर्देश
• कंप्यूटर-जनरेटेड प्रमाणपत्र
• टाइमस्टैम्प के साथ फुटर

विशेषताएं:
• एक-क्लिक PDF जनरेशन
• सीधे डिवाइस पर डाउनलोड करें
• प्रिंट-फ्रेंडली प्रारूप
• सरकारी मानक लेआउट
• द्विभाषी सामग्री समर्थन

4.5 भुगतान ट्रैकिंग
--------------------
भुगतान स्थिति:
• लंबित (लाल बैज)
• आंशिक (पीला बैज)
• भुगतान (हरा बैज)

संचालन:
• भुगतान रिकॉर्ड करें
• भुगतान स्थिति अपडेट करें
• भुगतान इतिहास देखें
• बकाया देय की गणना करें
• भुगतान रसीदें जनरेट करें

भुगतान जानकारी:
• भुगतान तिथि
• भुगतान विधि
• लेनदेन संदर्भ
• प्राप्त राशि
• रसीद संख्या

4.6 डैशबोर्ड
-------------
सांख्यिकी कार्ड:
• कुल संपत्तियां (आइकन के साथ गणना)
• पूर्ण भुगतान संपत्तियां (हरा संकेतक)
• लंबित बकाया (गणना के साथ लाल संकेतक)
• कुल राजस्व (स्वरूपित मुद्रा)

दृश्य विश्लेषण:

• भुगतान स्थिति अवलोकन (डोनट चार्ट)
• संग्रह रुझान (लाइन चार्ट)

त्वरित कार्रवाई:
• नई संपत्ति पंजीकृत करें
• बिल जनरेट करें
• रिपोर्ट देखें
• सिस्टम सेटिंग्स

नेविगेशन:
• आइकन और लेबल के साथ साइडबार
• ब्रेडक्रंब नेविगेशन
• खोज कार्यक्षमता

   - मूल्यांकन वर्ष-वार सारांश

2. संपत्ति रिपोर्ट:
   - संपत्ति प्रकार वितरण

   - क्षेत्र सांख्यिकी
   - मालिक जानकारी

3. संग्रह सारांश:
   - दैनिक संग्रह

4. भुगतान स्थिति रिपोर्ट:
   - भुगतान की गई संपत्तियों की सूची
   - लंबित संपत्तियों की सूची

विशेषताएं:

4.8 सेटिंग्स और कॉन्फ़िगरेशन

5.2 संपत्ति पंजीकरण
--------------------------
चरण 1: पंजीकरण पर जाएं
• साइडबार में "संपत्तियां" पर क्लिक करें
• "नई संपत्ति पंजीकृत करें" बटन पर क्लिक करें

चरण 2: संपत्ति विवरण भरें
• मालिक का नाम: संपत्ति मालिक का पूरा नाम
• पिता का नाम: मालिक के पिता का नाम
• मोबाइल नंबर: 10-अंकीय संपर्क नंबर
• पता: लैंडमार्क के साथ पूरा पता
• मकान नंबर: वैकल्पिक फ़ील्ड
• संपत्ति प्रकार: ड्रॉपडाउन से चुनें
• क्षेत्रफल (वर्ग फुट): संपत्ति क्षेत्र दर्ज करें
• स्थान: शहरी/अर्ध-शहरी/ग्रामीण चुनें

चरण 3: कर असाइन करें
• लागू करों को चेक करें:
  - संपत्ति कर (हमेशा लागू)
  - जल कर (यदि पानी कनेक्शन है)
  - स्वच्छता कर (यदि स्वच्छता सुविधा है)
  - प्रकाश कर (यदि सड़क प्रकाश व्यवस्था है)

चरण 4: संपत्ति सहेजें
• सभी विवरणों की समीक्षा करें
• "संपत्ति पंजीकृत करें" बटन पर क्लिक करें
• संपत्ति ID स्वतः जनरेट होगी
• पुष्टि संदेश दिखाई देता है

5.3 संपत्तियों का प्रबंधन
-----------------------
सभी संपत्तियां देखें:
• साइडबार में "संपत्तियां" पर क्लिक करें
• सभी पंजीकृत संपत्तियों की सूची देखें
• विशिष्ट संपत्ति खोजने के लिए खोज बॉक्स का उपयोग करें
• संपत्ति प्रकार/स्थान के लिए फ़िल्टर लागू करें

संपत्तियां खोजें:
• खोज बॉक्स में टाइप करें
• इसके द्वारा खोजें: मालिक का नाम, संपत्ति ID, मोबाइल नंबर
• परिणाम रियल-टाइम में अपडेट होते हैं

संपत्ति संपादित करें:
• संपत्ति पंक्ति पर "संपादित करें" बटन पर क्लिक करें
• आवश्यक फ़ील्ड अपडेट करें
• "परिवर्तन सहेजें" पर क्लिक करें
• पुष्टि संदेश दिखाई देता है

संपत्ति विवरण देखें:
• विस्तार करने के लिए संपत्ति पंक्ति पर क्लिक करें
• सभी संपत्ति जानकारी देखें
• असाइन किए गए कर देखें
• भुगतान इतिहास देखें

संपत्ति हटाएं:
• "हटाएं" बटन पर क्लिक करें (केवल व्यवस्थापक)
• विलोपन की पुष्टि करें
• संपत्ति सिस्टम से हटा दी गई

5.4 बिल जनरेट करना
--------------------
चरण 1: संपत्ति चुनें
• "संपत्तियां" पेज पर जाएं
• खोज का उपयोग करके संपत्ति खोजें
• "बिल जनरेट करें" बटन पर क्लिक करें

चरण 2: कर विवरण की समीक्षा करें
• कर विभाजन देखें
• गणना सटीकता की जांच करें
• संपत्ति विवरण सत्यापित करें
• भुगतान स्थिति की समीक्षा करें

चरण 3: PDF जनरेट करें
• "बिल जनरेट करें" बटन पर क्लिक करें
• PDF स्वचालित रूप से जनरेट होता है
• डाउनलोड तुरंत शुरू होता है
• वांछित स्थान पर सहेजें

चरण 4: बिल प्रिंट करें
• डाउनलोड किया गया PDF खोलें
• प्रिंट बटन पर क्लिक करें
• प्रिंटर चुनें
• A4 पेपर पर प्रिंट करें

5.5 भुगतान रिकॉर्ड करना
----------------------
चरण 1: संपत्ति तक पहुंच
• संपत्ति पर जाएं
• "भुगतान रिकॉर्ड करें" बटन पर क्लिक करें

चरण 2: भुगतान विवरण दर्ज करें
• भुगतान तिथि: तिथि चुनें
• प्राप्त राशि: राशि दर्ज करें
• भुगतान विधि: नकद/चेक/ऑनलाइन
• लेनदेन संदर्भ: यदि उपलब्ध हो तो दर्ज करें
• रसीद संख्या: स्वतः जनरेट

चरण 3: भुगतान सहेजें
• विवरणों की समीक्षा करें
• "भुगतान सहेजें" पर क्लिक करें
• भुगतान स्थिति स्वचालित रूप से अपडेट होती है
• रसीद जनरेट होती है

5.6 रिपोर्ट देखना
-------------------
चरण 1: रिपोर्ट तक पहुंच
• साइडबार में "रिपोर्ट" पर क्लिक करें
• मेनू से रिपोर्ट प्रकार चुनें

चरण 2: रिपोर्ट कॉन्फ़िगर करें
• तिथि सीमा चुनें
• फ़िल्टर चुनें
• समूहीकरण विकल्प लागू करें

चरण 3: देखें और निर्यात करें
• रिपोर्ट स्क्रीन पर प्रदर्शित होती है
• चार्ट और तालिकाएं देखें
• डाउनलोड करने के लिए "निर्यात" पर क्लिक करें
• हार्ड कॉपी के लिए "प्रिंट" पर क्लिक करें

6.1 उपयोगकर्ता प्रबंधन
-------------------
नए उपयोगकर्ता जोड़ना:
1. सेटिंग्स → उपयोगकर्ता पर जाएं
2. "नया उपयोगकर्ता जोड़ें" बटन पर क्लिक करें
3. उपयोगकर्ता विवरण दर्ज करें:
   - पूरा नाम
   - ईमेल पता
   - प्रारंभिक पासवर्ड
   - भूमिका (व्यवस्थापक/उपयोगकर्ता)
4. "उपयोगकर्ता बनाएं" पर क्लिक करें
5. उपयोगकर्ता को लॉगिन क्रेडेंशियल्स प्राप्त होते हैं

भूमिकाओं का प्रबंधन:
• व्यवस्थापक: पूर्ण सिस्टम पहुंच
  - संपत्तियों को पंजीकृत/संपादित/हटाएं
  - बिल जनरेट करें
  - भुगतान रिकॉर्ड करें
  - सभी रिपोर्ट देखें
  - सेटिंग्स कॉन्फ़िगर करें
  - उपयोगकर्ताओं का प्रबंधन करें

• उपयोगकर्ता: सीमित पहुंच
  - संपत्तियों को पंजीकृत/देखें
  - बिल जनरेट करें
  - रिपोर्ट देखें
  - हटा या सेटिंग्स कॉन्फ़िगर नहीं कर सकते

उपयोगकर्ताओं को निष्क्रिय करना:
1. सेटिंग्स → उपयोगकर्ता पर जाएं
2. सूची में उपयोगकर्ता खोजें
3. "निष्क्रिय करें" बटन पर क्लिक करें
4. कार्रवाई की पुष्टि करें
5. उपयोगकर्ता अब लॉगिन नहीं कर सकता

6.2 कर दर कॉन्फ़िगरेशन
--------------------------
संपत्ति कर दरें अपडेट करना:
1. सेटिंग्स → कर दरों पर जाएं
2. "संपत्ति कर" अनुभाग खोजें
3. दरें अपडेट करें:
   - शहरी दर (₹ प्रति वर्ग फुट)
   - अर्ध-शहरी दर (₹ प्रति वर्ग फुट)
   - ग्रामीण दर (₹ प्रति वर्ग फुट)
4. "परिवर्तन सहेजें" पर क्लिक करें

अन्य कर दरें अपडेट करना:
1. कर प्रकार चुनें (जल/स्वच्छता/प्रकाश)
2. प्रत्येक संपत्ति प्रकार के लिए दरें अपडेट करें:
   - आवासीय
   - वाणिज्यिक
   - औद्योगिक
   - कृषि
3. "परिवर्तन सहेजें" पर क्लिक करें

छूट और शुल्क सेट करना:
1. सेटिंग्स → कर कॉन्फ़िगरेशन पर जाएं
2. शीघ्र भुगतान छूट सेट करें (%)
3. विलंब भुगतान शुल्क सेट करें (% प्रति माह)
4. छूट समय सीमा तिथि सेट करें
5. "परिवर्तन सहेजें" पर क्लिक करें

6.3 पंचायत जानकारी
-------------------------
पंचायत विवरण अपडेट करना:
1. सेटिंग्स → पंचायत जानकारी पर जाएं
2. फ़ील्ड अपडेट करें:
   - पंचायत का नाम
   - पता
   - संपर्क नंबर
   - ईमेल पता
   - जिला
   - राज्य
3. लोगो अपलोड करें (भविष्य)
4. "परिवर्तन सहेजें" पर क्लिक करें

6.4 सिस्टम रखरखाव
----------------------
डेटाबेस बैकअप:
• Firebase द्वारा स्वचालित दैनिक बैकअप
• बैकअप डाउनलोड करें: Firebase Console → Database → Backup
• बैकअप को सुरक्षित रूप से स्टोर करें
• समय-समय पर रिस्टोर का परीक्षण करें

डेटा क्लीनअप:
• पुराने रिकॉर्ड आर्काइव करें (5+ वर्ष)
• डुप्लिकेट प्रविष्टियां हटाएं
• डेटा अखंडता सत्यापित करें
• स्थिरता जांच चलाएं

प्रदर्शन निगरानी:
• Firebase Console → Performance जांचें
• पेज लोड समय की निगरानी करें
• डेटाबेस क्वेरी की समीक्षा करें
• धीमे ऑपरेशन को अनुकूलित करें

सुरक्षा ऑडिट:
• उपयोगकर्ता एक्सेस लॉग की समीक्षा करें
• संदिग्ध गतिविधि की जांच करें
• सुरक्षा नियम अपडेट करें
• प्रमाणीकरण सेटिंग्स सत्यापित करें

• propertyId: string (अद्वितीय प्रदर्शन ID)

• createdBy: string (उपयोगकर्ता ID)

• assessmentYear: string (उदा., "2025-2026")

• amountPaid: number
• balanceDue: number
• dueDate: timestamp
• createdAt: timestamp
• updatedAt: timestamp

संग्रह: payments
दस्तावेज़ ID: स्वतः जनरेट
फ़ील्ड:
• propertyId: string (संपत्ति का संदर्भ)
• taxId: string (कर का संदर्भ)
• receiptNumber: string (RCT-YYYYMMDD-XXXXX)
• paymentDate: timestamp
• amount: number
• paymentMethod: string (नकद/चेक/ऑनलाइन)
• transactionRef: string (वैकल्पिक)
• recordedBy: string (उपयोगकर्ता ID)
• createdAt: timestamp

संग्रह: users

फ़ील्ड:
• uid: string (Firebase Auth UID)

• displayName: string

• lastLogin: timestamp
• isActive: boolean

संग्रह: settings
दस्तावेज़ ID: "panchayat"
फ़ील्ड:
• name: string
• address: string
• contactNumber: string
• email: string
• district: string
• state: string
• logoUrl: string (भविष्य)
• taxRates: object
  - property: object (urban, semiUrban, rural)
  - water: object (residential, commercial, industrial, agricultural)
  - sanitation: object (same structure)
  - lighting: object (same structure)
• discountRate: number
• penaltyRate: number
• discountDeadline: string (MM-DD format)
• updatedAt: timestamp

लक्षण: पंजीकरण फॉर्म त्रुटि दिखाता है

A. शब्दावली
--------------------
• मूल्यांकन वर्ष: वित्तीय वर्ष जिसके लिए कर की गणना की जाती है
• संपत्ति कर: संपत्ति क्षेत्र और स्थान पर आधारित कर
• जल कर: जल आपूर्ति कनेक्शन के लिए वार्षिक कर
• स्वच्छता कर: स्वच्छता सेवाओं के लिए वार्षिक कर
• प्रकाश कर: सड़क प्रकाश व्यवस्था के लिए वार्षिक कर
• शीघ्र भुगतान छूट: समय सीमा से पहले भुगतान के लिए छूट
• विलंब भुगतान शुल्क: अतिदेय भुगतान के लिए अतिरिक्त शुल्क
